Features / Commands

  • /register <username> <password> <password> - create account (no duplicate usernames)
  • /login <username> <password> - login account
  • /switch <username> - switch between logged in users
  • /logout - logout current user
  • /accounts - list logged in accounts
  • /send <username> "<message>" - send message to a user
  • /inbox - check inbox
  • /read <mail_id> - read a message
  • /delete <mail_id> - delete a mail
  • /clear - clear the inbox

Limitations

  • No GUI (intentionally)
  • No real-time updates
  • No notifications
  • No message editing

Note

There is no significant security in this application in v0.1. Although there is account verification and authentication, we do not guarantee protection from attacks.
