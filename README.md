cli chat application

current version : v0.1

v0.1

-register account by /register <username> <password> <confirm password> 
 cannot dubplicate a username
-login account by /login <username> <password>
-switch user : if you have logged in more than 1 users you can switch between them by /switch <username> , you don't have to login it everytime you wanna use it
-logout the current logged in account by /logout 
-check all the accounts logged in by /accounts
-send a message to any user /send <username> "<message>"
-check inbox by /inbox 
-read a message by /read <mail id>
-delete a mail by /delete <mail id>
-clear the inbox by /clear

NOTE
There is no significant security in this application in v0.1, altough there is account verification and authentication but we don't gurantee not being hacked.
